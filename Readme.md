**Бот поддержки**

Бот состоит из 2 сервисов:
1) Бот, который принимает сообщения от пользователя и отправляет ему ответ
2) Сайт для работников поддержки, в котором есть список чатов и возможность ответить каждому отдельно

Архитектура:
1) Клиент сайта на React, общается с бэкендом через webSocket. 
2) Бэкенд сайта на node.js, express. БД - PostgreSQL
3) Сам бот реализован на node.js, javascript, но при желании модно написать другого. Отправляет сообщения на сервис сайта через http запросы (POST).
